# Data Sources

This MVP uses official or internationally recognized public data sources.

## Live source used in the app

- World Bank Open Data / World Development Indicators API
  - https://data.worldbank.org/
  - API pattern: `https://api.worldbank.org/v2/country/{countries}/indicator/{indicator}?format=json`
  - The World Bank describes WDI as a compilation of high-quality, internationally comparable statistics from officially recognized sources.

## Official ASEAN reference source for next expansion

- ASEANstats Data Portal
  - https://data.aseanstats.org/
  - Relevant domains: ASEAN Key Statistics, Macroeconomic, Foreign Direct Investment Statistics, International Merchandise Trade Statistics, Statistics of International Trade in Services, Visitor Arrival Statistics, Price Statistics, Labour Statistics, Manufacturing Statistics.

## Planned additional official sources

- IMF Data Portal / World Economic Outlook and International Financial Statistics
  - https://data.imf.org/
  - Useful for projections, current account, fiscal indicators, inflation, unemployment, and macroeconomic forecasts.
- UN Comtrade
  - https://comtradeplus.un.org/
  - Useful for detailed merchandise trade by reporter, partner, and product.
